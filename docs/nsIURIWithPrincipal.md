# nsIURIWithPrincipal #
<code>  
nsIURIWithPrincipal is implemented by URIs which are associated with a  
specific principal.  
  
</code>
## Attributes ##

### principal ###
  
The principal associated with the resource returned when loading this  
uri.  
  

### principalUri ###
  
The uri for the principal.
