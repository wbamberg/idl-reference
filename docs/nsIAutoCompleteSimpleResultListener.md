# nsIAutoCompleteSimpleResultListener #

## Methods ##

### onValueRemoved ###
  
Dispatched after a value is removed from the result.  
@param aResult  
       The result from which aValue has been removed.  
@param aValue  
       The removed value.  
@param aRemoveFromDb  
       Whether the value should be removed from persistent storage as well.
