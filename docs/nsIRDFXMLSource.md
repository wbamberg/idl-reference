# nsIRDFXMLSource #

## Methods ##

### Serialize(aStream) ###
  
Serialize the contents of the datasource to aStream.  
@param aStream the output stream the will receive the  
  RDF/XML. Currently, the output stream need only  
  implement the |write()| method.
