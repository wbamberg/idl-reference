# nsIDOMCSSRuleList #
<pre>  
The nsIDOMCSSRuleList interface is a datatype for a list of CSS  
style rules in the Document Object Model.  
  
For more information on this interface please see  
http://www.w3.org/TR/DOM-Level-2-Style
