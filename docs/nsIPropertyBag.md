# nsIPropertyBag #

## Methods ##

### getProperty(name) ###
  
Get a property value for the given name.  
@throws NS_ERROR_FAILURE if a property with that name doesn't  
exist.  
  

## Attributes ##

### enumerator ###
  
Get a nsISimpleEnumerator whose elements are nsIProperty objects.
