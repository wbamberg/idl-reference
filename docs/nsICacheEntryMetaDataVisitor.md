# nsICacheEntryMetaDataVisitor #
  
Argument for nsICacheEntry.visitMetaData, provides access to all metadata  
keys and values stored on the entry.  
  

## Methods ##

### onMetaDataElement(key, value) ###
  
Called over each key / value pair.
