# extIConsole #
  
Interface that gives simplified access to the console  
  

## Methods ##

### log(aMsg) ###
  
Sends a given string to the console.  
@param   aMsg  
         The text to send to the console  
  

### open() ###
  
Opens the error console window. The console window  
is focused if already open.
