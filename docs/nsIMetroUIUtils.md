# nsIMetroUIUtils #

## Attributes ##

### currentPageURI ###
  
Obtains the current page URI  
  

### currentPageTitle ###
  
Obtains the current page title  
  

### hasSelectedContent ###
  
Determines if the browser has selected content  
  

### shareText ###
  
Determines the text that should be shared  
  

### shareHTML ###
  
Determines the HTML that should be shared
