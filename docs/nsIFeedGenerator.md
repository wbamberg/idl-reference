# nsIFeedGenerator #
<code>  
 An nsIFeedGenerator represents the software used to create a feed.  
  
</code>
## Attributes ##

### agent ###
  
The name of the software.  
  

### version ###
  
The version of the software.  
  

### uri ###
  
A URI associated with the software.
