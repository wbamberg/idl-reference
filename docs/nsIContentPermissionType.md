# nsIContentPermissionType #
  
 Interface provides the request type and its access.  
  

## Attributes ##

### type ###
  
 The type of the permission request, such as  
 "geolocation".  
  

### access ###
  
 The access of the permission request, such as  
 "read".  
  

### options ###
  
The array of available options.
