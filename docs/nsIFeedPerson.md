# nsIFeedPerson #
<pre>  
 An nsIFeedPerson represents an author or contributor of a feed.  
  
</pre>
## Attributes ##

### name ###
<pre>  
The name of the person.  
  
</pre>
### email ###
<pre>  
An email address associated with the person.  
  
</pre>
### uri ###
<pre>  
A URI associated with the person (e.g. a homepage).  
  
</pre>
