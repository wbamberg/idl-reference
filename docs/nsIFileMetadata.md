# nsIFileMetadata #
  
An interface that allows you to get some metadata like file size and  
file last modified time.  
  

## Attributes ##

### size ###
  
File size in bytes;  
  

### lastModified ###
  
File last modified time in milliseconds from midnight (00:00:00),  
January 1, 1970 Greenwich Mean Time (GMT).
