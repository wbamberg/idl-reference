# nsIParentChannel #
  
Implemented by chrome side of IPC protocols.  
  

## Methods ##

### setParentListener(listener) ###
  
Called to set the HttpChannelParentListener object (optional).  
  

### delete() ###
  
Called to invoke deletion of the IPC protocol.
