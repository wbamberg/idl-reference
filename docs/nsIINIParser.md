# nsIINIParser #

## Methods ##

### getSections() ###
  
Enumerates the [section]s available in the INI file.  
  

### getKeys(aSection) ###
  
Enumerates the keys available within a section.  
  

### getString(aSection, aKey) ###
  
Get the value of a string for a particular section and key.
